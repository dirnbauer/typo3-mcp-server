A TYPO3 extension that provides a `Model Context Protocol (MCP) <https://modelcontextprotocol.io/>`__
server, enabling AI assistants and LLMs to interact with TYPO3 pages,
content, and records through a standardized protocol.

**Key features:**

- Read and write TYPO3 content through MCP tools
- Full workspace integration -- all edits happen in workspaces
- Content translation support
- File browsing via FAL
- OAuth 2.1 authentication with PKCE
- Compatible with TYPO3 v13.4 and v14
